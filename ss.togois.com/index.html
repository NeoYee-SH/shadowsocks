3. 安装基本工具

[root@vultr_test ~]# yum -y install net-tools vim
[root@vultr_test ~]# wget --no-check-certificate https://bootstrap.pypa.io/get-pip.py
[root@vultr_test ~]# python2.7 get-pip.py

4. 安装ss并配置

#安装
[root@vultr_test ~]# pip install shadowsocks

#配置
[root@vultr_test ~]# vim /etc/shadowsocks.json
---
{
    "server":"0.0.0.0",
    "server_port":8765,
    "password":"bd87d3600c6baffe20f0a03ebf17985f",
    "timeout":300,
    "method":"rc4-md5"
}
---

#启动
[root@vultr_test ~]# ssserver -c /etc/shadowsocks.json -d start 
[root@vultr_test ~]# netstat -anpt |grep python
tcp        0      0 0.0.0.0:8765            0.0.0.0:*               LISTEN      1162/python2.7  

#关闭防火墙
[root@vultr_test ~]# systemctl stop firewalld
[root@vultr_test ~]# systemctl disable firewalld

客户端

1. Mac上

https://github.com/shadowsocks/ShadowsocksX-NG/releases 可选版本。

下载后启动： 单击小飞机，选择服务器->服务器设置，进入设置界面：

添加一个服务器配置，对应的参数是服务端/etc/shadowsocks.json里的参数，填写好相关选项，确定就OK了。

2. Windows上

下载 ：https://github.com/shadowsocks/shadowsocks-windows/releases

双击启动后， 在桌面右下角可以看到小飞机：

鼠标放在小飞机上面，然后右键->服务器->编辑服务器->添加：

对应的参数是服务端/etc/shadowsocks.json里的参数，设置一个代理端口，通常是1080。

这样windows上的客户端就安装成功，但是现在还是不能访问外网，需要使用浏览器的扩展插件。

谷歌浏览器下载扩展程序SwitchyOmega.crx； 打开谷歌扩展程序页面chrome://extensions/； 把SwitchyOmega.crx文件拖到扩展程序页面添加，

添加后通常会自动跳转到设置界面：

可以直接跳过教程，也可以再看看。 选择代理，设置为

最后点击应用选项

保存成功之后，看到浏览器右上角的logo：

单机，选择proxy代理模式就可以了：
